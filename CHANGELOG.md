# CHANGELOG

## v0.2.1

### 🚀 优化

- **客户端运行更轻量稳定**：网卡列表读取与文件选择不再借助系统 PowerShell 完成，改为原生实现，运行时不再启动外部子进程，降低了资源占用，也避免了部分环境下安全软件对外部子进程的误报与拦截。

### 🐛 Bug 修复

- **修复长文本消息溢出气泡的问题**：发送长串文字或长链接时不再撑破消息气泡，会自动按宽度换行；同时保留了你输入时的换行与空格格式。

## v0.2.0

### ✨ 新功能

- **一键分享码快速加群**：群组创建者可在分享对话框生成分享码，其中包含群组名称、群组 ID 与所有可连接的中继地址；新设备粘贴分享码即可自动填入群组信息，并自动探测可用的中继地址与本机网卡，无需手动逐项填写 IP 与端口。
- **客户端自动更新**：Windows 客户端接入在线更新能力，启动时自动检查新版本，也可手动触发检查；发现新版本后可查看版本号、发布时间与更新说明，确认后自动下载并安装，无需手动下载安装包。

### 🚀 优化

- **发布版隐藏控制台窗口**：正式发布版本运行时不再弹出黑色控制台窗口，仅显示主程序界面，视觉更清爽。

## v0.1.0

### ✨ 新功能

- **局域网多跳 Mesh 通信**：支持同一群组内任意数量设备加入，消息可通过中间设备多跳转发到达目标，无需全网互相连通，无需互联网与中心服务器。
- **群组创建与加入**：Windows 端可创建群组（作为中继）或加入已有群组（作为叶子），支持局域网自动发现可用群组，也可手动输入 IP 与端口加入。
- **群聊消息**：群发消息通过多跳洪泛到达群组内所有成员。
- **单聊消息**：通过路径发现建立定向转发路径，仅沿该路径传递消息，不做全群广播。
- **文件传输**：大文件自动分片发送，支持断点续传（中断后只补传缺失分片），接收完成后自动校验文件完整性；发送时携带文件名，接收方可直接保存。
- **成员状态管理**：实时展示群组成员上线/下线状态、直连与多跳可达标记。
- **连接保活**：邻居间定时心跳检测，超时自动判定离线；断线后自动重连。
- **多网卡选择**：创建或加入群组时可指定本机网卡，避免系统默认路由选错网络接口。
- **Windows 桌面客户端**：基于 Tauri 的原生客户端，提供群组管理、消息收发、文件传输、网络配置等完整界面。
- **文件接收与另存**：接收文件后可选择保存位置，支持多网卡环境下广播发现。
- **中文网卡名称兼容**：修复了 Windows 系统中网络接口名称含中文时的显示问题。
- **安装包分发**：支持打包为 NSIS 单文件安装程序，通过 LTO 与符号裁剪优化体积。
